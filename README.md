# Security Projects Portfolio

A collection of Python-based cybersecurity and network analysis projects I built independently and through academic work at Eastern Washington University.

## Projects

### Great Cannon
Analyzes packet capture data to identify traffic patterns associated with the Great Cannon and Great Firewall using known behavioral signatures.

- Focus: traffic analysis, censorship tooling detection, packet inspection
- Files: `lab6.py`, `both_sidechannel.tcpdump`

### Host Discovery
Passively discovers hosts from packet capture files by extracting unique IP addresses and preparing them for follow-up scanning.

- Focus: passive reconnaissance, packet parsing, network enumeration
- Files: `host_discovery.py`, `smtp.pcap`

## Skills Demonstrated

- Passive network reconnaissance
- Traffic inspection
- Cybersecurity problem solving
