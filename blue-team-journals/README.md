# Blue Team Journals

This directory contains my cybersecurity learning journal focused on blue team and defensive security topics.

I created this section to document what I study, practice, and learn as I continue developing skills relevant to SOC analyst, threat detection, and incident response roles. These entries serve as a record of continuous improvement, technical curiosity, and hands-on learning.

## Purpose

The goal of this journal is to:

- Document my progress in cybersecurity
- Reinforce concepts through written reflection
- Practice clear and organized technical communication
- Track hands-on exercises and lab work
- Build a public portfolio of consistent learning

## Topics Covered

Entries in this directory may include:

- CySA+ study notes
- Blue team concepts and terminology
- Log analysis and monitoring practice
- Incident response workflows
- Detection-focused observations
- Security tools and command usage
- Lab documentation and lessons learned

## Format

Each entry may include some or all of the following:

- Topics studied
- Summary of concepts learned
- Hands-on tasks performed
- Key findings or takeaways
- Questions for follow-up
- Next steps

## Professional Focus

This journal reflects my interest in defensive security and my effort to build practical, job-relevant skills through consistent study and documentation.

## Disclaimer

All content here is written for educational and portfolio purposes. Notes are summarized in my own words and are intended to reflect my understanding, practice, and growth in cybersecurity.
